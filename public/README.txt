DARK MESSENGER SERVER

Что внутри:
- отдельный Node.js сервер
- регистрация и вход
- личные чаты между пользователями
- сообщения в реальном времени через Socket.IO
- база SQLite: messenger.db создается автоматически

Как запустить на ПК:
1. Установи Node.js с сайта nodejs.org
2. Распакуй архив
3. Открой папку dark_messenger_server
4. В адресной строке проводника напиши cmd и нажми Enter
5. Выполни команды:

npm install
npm start

6. На этом ПК открой:
http://localhost:3000

7. Для другого устройства в той же Wi-Fi сети используй адрес, который покажет консоль:
http://192.168.x.x:3000

Как сделать как Telegram через интернет:
- этот сервер нужно залить на VPS/хостинг, например Render, Railway или любой VPS
- открыть порт 3000 или настроить домен через nginx
- в server.js обязательно поменять JWT_SECRET на свой длинный секрет

Важно:
Это учебный мессенджер. Для настоящего продакшена нужны HTTPS, нормальная защита, восстановление пароля, шифрование, модерация, резервные копии.
